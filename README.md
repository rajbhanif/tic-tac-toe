# Tic Tac Toe

Un jeu Tic Tac Toe classique développé en Kotlin pour Android avec intelligence artificielle stratégique. Cette application offre une expérience de jeu fluide avec une interface élégante.

## Aperçu du Projet

Jeu Tic Tac Toe où le joueur affronte une IA intelligente dans une interface moderne violet/noir/blanc. Le jeu alterne entre le joueur et l'ordinateur.

## Fonctionnalités

### Contrôle de Jeu Intelligent
- Tour Alterné - Joueur (X) vs IA (O)
- Reconnaissance Instantanée - Détection automatique des victoires
- Système de Feedback :
  - Lignes gagnantes en violet clair
  - Dialogues de félicitations personnalisés
  - Match nul détecté automatiquement
- Gestion des Noms - Joueur peut entrer son nom personnalisé
- Base de Logique Stratégique - IA prend des décisions intelligentes

### Caractéristiques Techniques
- Intégration Complète Android - Activités, Layouts XML, Ressources
- Performance Optimisée - Fonctionne fluidement sur mobile
- Interface Utilisateur Intuitive - Navigation simple et claire
- Personnalisation Flexible - Nom ajustable selon le joueur

## Comment Utiliser

### Processus de Jeu :
1. Démarrage : Entrer votre nom dans l'écran d'accueil
2. Votre Tour : Cliquer sur une case vide pour placer X
3. Tour IA : L'ordinateur place automatiquement O (500ms de délai)
4. Résultat :
   - Victoire : 3 symboles alignés - ligne mise en évidence
   - Match Nul : Grille pleine sans gagnant
5. Recommencer : Appuyer sur "Recommencer le jeu"
